Encryption Standards

  • AES-256-GCM: Industry-standard authenticated encryption
  • scrypt KDF: Memory-hard password hashing (N=16384, r=8, p=1)
  • Unique Salt & IV: Each file gets its own salt and initialization vector
  • Authentication Tags: Prevents tampering and ensures data integrity

Privacy Guarantees

  • Local Key Derivation: Passwords never leave your machine
  • Zero-Knowledge: We cannot access your encrypted files
  • Memory Safety: Keys are zeroed from memory after use
  • No Network: All operations happen completely offline
